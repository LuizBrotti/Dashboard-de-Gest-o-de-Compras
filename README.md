
# 📊 Análise de Dados de Compras e Fornecedores

## 📌 Sobre o Projeto

Este projeto apresenta uma análise de dados baseada em um banco de dados de compras e fornecedores. O objetivo é identificar padrões de compra, avaliar o desempenho dos fornecedores e analisar possíveis problemas no processo logístico, como atrasos nas entregas.

A análise foi realizada por meio de visualizações gráficas e interpretação dos dados, permitindo extrair insights relevantes para a gestão de compras e supply chain.

---

## 🎯 Objetivos da Análise

* Identificar o volume total de compras por comprador.
* Analisar quais fornecedores representam maior volume financeiro.
* Avaliar os gastos por tipo de matéria-prima.
* Identificar a porcentagem de pedidos entregues no prazo e em atraso.
* Comparar o tempo médio de entrega entre fornecedores.

---

## 📈 Análises Realizadas

### 1️⃣ Valor total por comprador

Análise do volume de compras realizadas por cada comprador, identificando concentração de compras entre os responsáveis.

### 2️⃣ Valor total comprado por fornecedor

Avaliação do total adquirido de cada fornecedor para entender quais representam maior impacto financeiro.

### 3️⃣ Valor total por matéria-prima

Identificação das matérias-primas com maior valor de compra, permitindo visualizar quais insumos geram maior custo.

### 4️⃣ Porcentagem de pedidos em atraso

Análise da proporção de pedidos entregues no prazo em comparação com pedidos atrasados.

### 5️⃣ Tempo médio de entrega por fornecedor

Comparação do tempo médio de entrega entre fornecedores para avaliar desempenho logístico.

---

## 🔎 Principais Insights

* O volume de compras está concentrado em poucos compradores.
* Alguns fornecedores representam grande parte do valor total de compras.
* Certas matérias-primas possuem impacto maior no custo total.
* Foi identificado um alto índice de atrasos nas entregas, indicando possíveis gargalos logísticos.
* Há diferença no tempo médio de entrega entre fornecedores.
